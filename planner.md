# Weather App

A web application to help user to see Weather data from your location.

Intends to be international.

Is an upgrade from the challenge "#boraCodar" from RocketSeat.

Runs in the browser.

## Tech stack

* HTML

* CSS
  * SCSS to generated the minified CSS

* Javascript/Typescript
  * Tools
    * ESLint
    * JSHint
    * NodeJS
  * Libraries
    * i18n
    * React
    * BackboneJS

* [Free Weather API](https://www.weatherapi.com/)
